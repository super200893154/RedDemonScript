### 非功能需求（NFR）

#### 可靠性
| 编号 | 需求 |
|------|------|
| NFR1 | 系统可以连续运行 ≥ 12 小时不崩溃 |
| NFR2 | 系统内存占用在长时间运行中保持稳定（无内存泄漏） |
| NFR3 | 三线程协调运行无死锁、无竞态条件 |
| NFR4 | 异常恢复成功率 ≥ 90% |
| NFR5 | 系统可以在游戏闪退后 60 秒内完成恢复 |

#### 性能
| 编号 | 需求 |
|------|------|
| NFR6 | 系统内存占用 < 100MB（不含游戏进程） |
| NFR7 | 系统CPU占用 < 5%（不含游戏进程和DLL操作） |
| NFR8 | UI 操作响应时间 < 200ms |
| NFR9 | 日志写入延迟 < 50ms |

#### 集成
| 编号 | 需求 |
|------|------|
| NFR10 | 系统可以与 Sandboxie Plus 5.55+ 版本兼容 |
| NFR11 | 外部 DLL 可替换，无需重新编译主程序 |
| NFR12 | 配置文件格式向后兼容（v1.x 配置可在 v1.y 中使用） |

#### 可维护性
| 编号 | 需求 |
|------|------|
| NFR13 | 代码模块化，用户接口与框架代码分离 |
| NFR14 | 日志格式标准化，支持问题快速定位 |
| NFR15 | 配置文件采用 JSON 格式，人类可读可编辑 |

**NFR 总数：15**
