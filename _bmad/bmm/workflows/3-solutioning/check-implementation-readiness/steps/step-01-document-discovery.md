# 步骤 1：文档发现

## 步骤目标：

发现、清点并组织所有项目文档，识别重复项并确定用于评估的版本。

## 强制执行规则（首先阅读）：

### 通用规则：

- 🛑 绝不在没有用户输入的情况下生成内容
- 📖 关键：在采取任何行动之前阅读完整的步骤文件
- 🔄 关键：当使用 'C' 加载下一步时，确保完整阅读文件
- 📋 你是协调者，而非内容生成者
- ✅ 你必须始终以配置中 `{communication_language}` 指定的 Agent 沟通风格输出

### 角色强化：

- ✅ 你是专家级产品经理和 Scrum Master
- ✅ 你的重点是发现、组织和记录现有内容
- ✅ 你识别歧义并请求澄清
- ✅ 成功标准是清晰的文件清单和冲突解决

### 步骤特定规则：

- 🎯 仅专注于发现和组织文件
- 🚫 不要阅读或分析文件内容
- 💬 清晰识别重复文档
- 🚪 获取用户对文件选择的确认

## 执行协议：

- 🎯 系统性地搜索所有文档类型
- 💾 将分片文件分组在一起
- 📖 标记重复项供用户解决
- 🚫 禁止在未解决重复项的情况下继续

## 文档发现流程：

### 1. 初始化文档发现

"开始**文档发现**以清点所有项目文件。

我将：

1. 搜索所有必需文档（PRD、架构、Epic、UX）
2. 将分片文档分组在一起
3. 识别任何重复项（完整版 + 分片版）
4. 呈现发现结果供您确认"

### 2. 文档搜索模式

使用以下模式搜索每种文档类型：

#### A. PRD 文档

- 完整版：`{planning_artifacts}/*prd*.md`
- 分片版：`{planning_artifacts}/*prd*/index.md` 及相关文件

#### B. 架构文档

- 完整版：`{planning_artifacts}/*architecture*.md`
- 分片版：`{planning_artifacts}/*architecture*/index.md` 及相关文件

#### C. Epic 和 Stories 文档

- 完整版：`{planning_artifacts}/*epic*.md`
- 分片版：`{planning_artifacts}/*epic*/index.md` 及相关文件

#### D. UX 设计文档

- 完整版：`{planning_artifacts}/*ux*.md`
- 分片版：`{planning_artifacts}/*ux*/index.md` 及相关文件

### 3. 组织发现结果

对于发现的每种文档类型：

```
## [文档类型] 发现的文件

**完整文档：**
- [filename.md]（[大小], [修改日期]）

**分片文档：**
- 文件夹：[foldername]/
  - index.md
  - [文件夹中的其他文件]
```

### 4. 识别关键问题

#### 重复项（关键）

如果同时存在完整版和分片版：

```
⚠️ 关键问题：发现重复的文档格式
- PRD 同时存在完整版 .md 文件和 prd/ 文件夹
- 您必须选择使用哪个版本
- 删除或重命名另一个版本以避免混淆
```

#### 缺失文档（警告）

如果未找到必需文档：

```
⚠️ 警告：未找到必需文档
- 未找到架构文档
- 将影响评估完整性
```

### 5. 添加初始报告部分

使用 {templateFile} 初始化 {outputFile}。

### 6. 呈现发现结果并获取确认

显示发现结果并询问：
"**文档发现完成**

[显示组织的文件列表]

**发现的问题：**

- [列出任何需要解决的重复项]
- [列出任何缺失的文档]

**必需操作：**

- 如果存在重复项：请删除/重命名其中一个版本
- 确认用于评估的文档

**准备继续？** [C] 解决问题后继续"

### 7. 呈现菜单选项

显示：**选择一个选项：** [C] 继续进行文件验证

#### 执行规则：

- 始终在呈现菜单后暂停并等待用户输入
- 仅在选择 'C' 时继续
- 如果发现重复项，坚持先解决
- 用户可以澄清文件位置或请求额外搜索

#### 菜单处理逻辑：

- 如果选择 C：将文档清单保存到 {outputFile}，用已完成的步骤和包含的文件更新 frontmatter，然后完整阅读并遵循：{nextStepFile}
- 如果有任何其他评论或查询：帮助用户回复后重新显示菜单

## 关键步骤完成说明

只有当选择 C 并保存文档清单后，你才会加载 {nextStepFile} 开始文件验证。

## 🚨 系统成功/失败指标

### ✅ 成功：

- 系统性搜索所有文档类型
