# 项目头脑风暴上下文模板

## 项目关注领域

此头脑风暴会议专注于软件和产品开发考虑事项：

### 关键探索领域

- **用户问题和痛点** - 用户面临哪些挑战？
- **功能想法和能力** - 产品可以做什么？
- **技术方法** - 我们可能如何构建它？
- **用户体验** - 用户将如何与之交互？
- **商业模式和价值** - 它如何创造价值？
- **市场差异化** - 什么使它独特？
- **技术风险和挑战** - 可能出什么问题？
- **成功指标** - 我们将如何衡量成功？

### 与项目工作流集成

头脑风暴结果可能输入到：

- 产品简介，用于初始产品愿景
- PRD，用于详细需求
- 技术规格，用于架构计划
- 研究活动，用于验证需求
